# hello-apollo

Apollo + TypeScript + Codegen + React

## Requirement

- Node.js ^12.16.1
- Yarn ^1.22.4

## Installation

```bash
yarn install
```

## Usage

```bash
# Server
yarn server start  # http://localhost:4000 が起動する

# Client
yarn client setup  # ES Modules コードが生成される
yarn client start  # http://localhost:3000 が起動する
```
